Welcome to React Native

What is React Native?

This is a best-in-class JavaScript library for creating native apps for Android and iOS

Who created it?

Facebook in 2015

In core: JavaScript, Java, C++, Objective-C

Knowledge:

1. Good knowledge of JavaScript - ES6+

2. Basic knowledge of React

Link to official website:

https://reactnative.dev

Advantages:

1. Cross-platform development (Android + IOS)

2. Easy and understandable to use

3. Performance

4. Community

Setting up the development environment:

Two CLI for React Native:

1. Expo Go CLI (better for new mobile developers)

https://expo.dev

2. React Native CLI

Installing Expo Go CLI:

yarn global add expo-cli

Generating an empty project:

yarn create expo-app project-name

Installing additional dependencies:

          
          npx expo install react-native-web@~0.18.9
          react-dom@18.1.0 @expo/webpack-config@^0.17.2

to run your development in a browser

Now we can run our project in a browser:

yarn web
